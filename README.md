### Proyectos

**[Olimpiadas 2025](https://github.com/JesusBaute/Olimpiadas_2025)** — Plataforma de turismo full-stack desarrollada en equipo para una olimpíada escolar. Flujo completo: registro, login, carrito y pago real con Mercado Pago. Me ocupé del backend entero: diseño de la base de datos relacional en MySQL, sistema de autenticación con JWT y cookies, middleware de roles (usuario/admin) y la integración con la API de MP. Todo containerizado con Docker y Nginx.

**[Bearlife](https://github.com/JesusBaute/BearLife)** · [▶ demo](https://bearlife.vercel.app) — Videojuego 2D de supervivencia en mundo abierto (5000×5000px) con Phaser 3, deployado en Vercel. Desarrollé la generación procedural del mapa, el sistema de inventario con stack de ítems, la lógica de vida/daño/curación y la integración general de mecánicas. Fue el primer proyecto donde tuve que pensar en arquitectura de sistemas, no solo en código que funcione.
